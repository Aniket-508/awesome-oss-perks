---
title: ossperks categories
description: Alle verfügbaren Programmkategorien und ihre IDs auflisten.
---

## Verwendung

```
ossperks categories [optionen]
```

## Optionen

| Flag         | Beschreibung        |
| ------------ | ------------------- |
| `--json`     | Rohes JSON ausgeben |
| `-h, --help` | Hilfe anzeigen      |

## Beispiele

```bash

# Alle Kategorien auflisten

ossperks categories

# JSON-Ausgabe

ossperks categories --json
```

## Ausgabe

```
13 Kategorien
──────────────

  ID               NAME                    COUNT
  hosting          Hosting & Deployment       12
  devtools         Developer Tools             8
  ...
```

Die Spalte **COUNT** gibt die Anzahl der Programme in dieser Kategorie an (Beispielwerte).
