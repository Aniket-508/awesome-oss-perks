## Verwendung

```
ossperks open
```

## Beschreibung

Öffnet [ossperks.com](https://www.ossperks.com) in Ihrem Standardbrowser. Verwendet den plattformspezifischen Befehl (`open` unter macOS, `xdg-open` unter Linux, `start` unter Windows).

Wenn der Browser nicht gestartet werden kann, wird die URL stattdessen im Terminal ausgegeben.

## Beispiele

```bash

# Website öffnen

ossperks open
```
