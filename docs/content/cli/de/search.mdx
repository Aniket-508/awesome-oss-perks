---
title: ossperks search
description: Durchsuchen Sie OSS-Vergünstigungsprogramme nach Schlüsselwörtern
  in Namen, Beschreibungen und Anbietern.
---

## Verwendung

```
ossperks search <Suchbegriff> [Optionen]
```

## Argumente

| Argument  | Beschreibung                                                                       |
| --------- | ---------------------------------------------------------------------------------- |
| `<query>` | Suchbegriff (stimmt überein mit Name, ID, Beschreibung, Anbieter, Kategorie, Tags oder Vorteilen) |

## Optionen

| Flag                    | Beschreibung                                   |
| ----------------------- | ---------------------------------------------- |
| `-c, --category <slug>` | Suche auf eine bestimmte Kategorie beschränken |
| `--json`                | Rohes JSON ausgeben                            |
| `-h, --help`            | Hilfe anzeigen                                 |

## Beispiele

```bash

# Suche nach "hosting"

ossperks search hosting

# Suche nach "credits" in der Kategorie devtools

ossperks search credits --category devtools

# JSON-Ausgabe

ossperks search vercel --json
```

## Ausgabe

```
Suchergebnisse für "hosting" — 4 Programme
────────────────────────────────────────────

  ID            NAME                            URL
  vercel        Vercel for Open Source          https://ossperks.com/programs/vercel
  netlify       Netlify Open Source Plan        https://ossperks.com/programs/netlify
  digitalocean  DigitalOcean Open Source        https://ossperks.com/programs/digitalocean
  cloudflare    Cloudflare for Open Source      https://ossperks.com/programs/cloudflare
```
