---
title: ossperks list
description: Alle verfügbaren OSS-Vorteilsprogramme auflisten, optional nach
  Kategorie gefiltert.
---

## Verwendung

```
ossperks list [options]
```

**Alias:** `ossperks ls`

## Optionen

| Flag                    | Beschreibung                              |
| ----------------------- | ---------------------------------------- |
| `-c, --category <id>` | Nach Kategorie-ID filtern (z. B. `hosting`) |
| `--json`                | Rohe JSON-Ausgabe                          |
| `-h, --help`            | Hilfe anzeigen                                |

## Beispiele

```bash

# Alle Programme auflisten

ossperks list

# Nur Hosting-Programme filtern

ossperks list --category hosting

# Maschinenlesbare JSON-Ausgabe

ossperks list --json
```

## Ausgabe

```
48 Programme
────────────

  ID                NAME                                          URL
  1password         1Password for Open Source Projects            https://ossperks.com/programs/1password
  algolia           Algolia for Open Source                         https://ossperks.com/programs/algolia
  anthropic-claude  Anthropic Claude for Open Source                https://ossperks.com/programs/anthropic-claude
  ...
```
