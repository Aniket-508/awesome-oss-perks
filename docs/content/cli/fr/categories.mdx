---
title: ossperks categories
description: Liste toutes les catégories de programmes disponibles et leurs identifiants.
---

## Utilisation

```
ossperks categories [options]
```

**Alias:** `ossperks cats`

## Options

| Drapeau      | Description           |
| ------------ | --------------------- |
| `--json`     | Afficher le JSON brut |
| `-h, --help` | Afficher l'aide       |

## Exemples

```bash

# Lister toutes les catégories

ossperks categories

# Sortie JSON

ossperks categories --json
```

## Sortie

```
13 catégories
─────────────

  ID               NOM                     NOMBRE
  hosting          Hébergement & Déploiement  12
  devtools         Outils de Développement     8
  ...
```

La colonne **NOMBRE** indique le nombre de programmes dans cette catégorie (valeurs d'exemple).
