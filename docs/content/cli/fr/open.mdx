## Utilisation

```
ossperks open
```

## Description

Ouvre [ossperks.com](https://www.ossperks.com) dans votre navigateur par défaut. Utilise la commande appropriée à la plateforme (`open` sur macOS, `xdg-open` sur Linux, `start` sur Windows).

Si le navigateur ne peut pas être lancé, l'URL est affichée dans le terminal à la place.

## Exemples

```bash

# Ouvrir le site web

ossperks open
```
