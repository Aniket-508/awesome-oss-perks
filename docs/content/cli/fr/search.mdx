---
title: ossperks search
description: Recherchez des programmes d'avantages OSS par mot-clé parmi les
  noms, descriptions, fournisseurs, tags et avantages.
---

## Utilisation

```
ossperks search <query> [options]
```

**Alias :** `ossperks find`, `ossperks s`

## Arguments

| Argument  | Description                                                                                          |
| --------- | ---------------------------------------------------------------------------------------------------- |
| `<query>` | Terme de recherche (correspond au nom, slug, description, fournisseur, catégorie, tags ou avantages) |

## Options

| Option       | Description       |
| ------------ | ----------------- |
| `--json`     | Sortie JSON brute |
| `-h, --help` | Afficher l'aide   |

## Exemples

```bash

# Rechercher « hosting »

ossperks search hosting

# Utilisation de l'alias court

ossperks s credits

# Sortie JSON

ossperks search vercel --json
```

## Résultat

```
Résultats de recherche pour "hosting" — 4 programmes
────────────────────────────────────────────

  vercel                 Vercel for Open Source
  netlify                Netlify OSS Plan
  digitalocean           DigitalOcean Open Source
  cloudflare             Cloudflare for Open Source
```
