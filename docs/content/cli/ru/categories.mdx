title: ossperks categories
description: Список всех доступных категорий программ и их идентификаторов.
---

## Использование

```
ossperks categories [options]
```

**Псевдоним:** `ossperks cats`

## Опции

| Флаг         | Описание                    |
| ------------ | --------------------------- |
| `--json`     | Вывести необработанный JSON |
| `-h, --help` | Показать справку            |

## Примеры

```bash

# Список всех категорий

ossperks categories

# Вывод в формате JSON

ossperks categories --json
```

## Вывод

```
13 categories
─────────────

  ID               NAME                    COUNT
  hosting          Hosting & Deployment       12
  devtools         Developer Tools             8
  ...
```

Столбец **COUNT** — это количество программ в данной категории (примерные значения).
