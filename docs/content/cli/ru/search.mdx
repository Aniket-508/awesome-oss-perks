---
title: ossperks search
description: Поиск программ льгот для OSS по ключевому слову среди названий,
  описаний и провайдеров.
---

## Использование

```
ossperks search <query> [options]
```

## Аргументы

| Аргумент  | Описание                                                                       |
| --------- | ------------------------------------------------------------------------------ |
| `<query>` | Поисковый запрос (совпадает с именем, id, описанием, провайдером, категорией, тегами или преимуществами) |

## Опции

| Флаг                    | Описание                                 |
| ----------------------- | ---------------------------------------- |
| `-c, --category <slug>` | Ограничить поиск определённой категорией |
| `--json`                | Вывести необработанный JSON              |
| `-h, --help`            | Показать справку                         |

## Примеры

```bash

# Поиск по запросу "hosting"

ossperks search hosting

# Поиск по запросу "credits" в категории devtools

ossperks search credits --category devtools

# Вывод в формате JSON

ossperks search vercel --json
```
