## Использование

```
ossperks show <id> [опции]
```

**Псевдоним:** `ossperks info`

## Аргументы

| Аргумент | Описание                                   |
| -------- | ------------------------------------------ |
| `<id>`   | Идентификатор программы (например, `vercel`, `github-pages`) |

## Опции

| Флаг         | Описание                    |
| ------------ | --------------------------- |
| `--json`     | Вывести необработанный JSON |
| `-h, --help` | Показать справку            |

## Примеры

```bash

# Показать программу Vercel

ossperks show vercel

# Машиночитаемый JSON

ossperks show vercel --json
```
