## Использование

```
ossperks open
```

## Описание

Открывает [ossperks.com](https://www.ossperks.com) в вашем браузере по умолчанию. Использует команду, соответствующую платформе (`open` на macOS, `xdg-open` на Linux, `start` на Windows).

Если браузер не удается запустить, URL выводится в терминал.

## Примеры

```bash

# Открыть веб-сайт

ossperks open
```
