---
title: ossperks list
description: Вывести список всех доступных программ льгот для OSS-проектов с
  возможностью фильтрации по категории.
---

## Использование

```
ossperks list [options]
```

## Опции

| Флаг                    | Описание                                 |
| ----------------------- | ---------------------------------------- |
| `-c, --category <id>` | Фильтр по идентификатору категории (например, `hosting`) |
| `--json`                | Вывод в формате JSON                     |
| `-h, --help`            | Показать справку                         |

## Примеры

```bash

# Показать все программы

ossperks list

# Отфильтровать только программы хостинга

ossperks list --category hosting

# Вывод в машиночитаемом формате JSON

ossperks list --json
```
