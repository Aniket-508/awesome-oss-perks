---
title: ossperks categories
description: 利用可能なすべてのプログラムカテゴリとそのIDを一覧表示します。
---

## 使用方法

```
ossperks categories [options]
```

**エイリアス:** `ossperks cats`

## オプション

| フラグ       | 説明           |
| ------------ | -------------- |
| `--json`     | 生のJSONを出力 |
| `-h, --help` | ヘルプを表示   |

## 例

```bash

# 全カテゴリーを一覧表示

ossperks categories

# JSON出力

ossperks categories --json
```

## 出力

```
13 categories
─────────────

  ID               NAME                    COUNT
  hosting          Hosting & Deployment       12
  devtools         Developer Tools             8
  ...
```

**COUNT** 列は、そのカテゴリ内のプログラム数を示します（例示値）。
