---
title: ossperks list
description: 利用可能なすべてのOSS特典プログラムを一覧表示し、オプションでカテゴリでフィルタリングします。
---

## 使用方法

```
ossperks list [options]
```

**エイリアス:** `ossperks ls`

## オプション

| フラグ                  | 説明                                       |
| ----------------------- | ------------------------------------------ |
| `-c, --category <slug>` | カテゴリスラグでフィルター (例: `hosting`) |
| `--json`                | 生のJSONを出力                             |
| `-h, --help`            | ヘルプを表示                               |

## 使用例

```bash

# すべてのプログラムを一覧表示

ossperks list

# ホスティングプログラムのみにフィルター

ossperks list --category hosting

# 機械可読なJSON出力

ossperks list --json
```

## 出力

```
OSS Perks — 48プログラム
────────────────────────────────

  1password              1Password for Open Source Projects
  algolia                Algolia for Open Source
  anthropic-claude       Anthropic Claude for Open Source
  ...
```
