---
title: ossperks categories
description: 列出所有可用的程序分类及其 ID。
---

## 用法

```
ossperks categories [options]
```

**别名：** `ossperks cats`

## 选项

| 标志         | 描述          |
| ------------ | ------------- |
| `--json`     | 输出原始 JSON |
| `-h, --help` | 显示帮助      |

## 示例

```bash

# 列出所有类别

ossperks categories

# JSON 输出

ossperks categories --json
```

## 输出

```
13 categories
─────────────

  ID               NAME                    COUNT
  hosting          Hosting & Deployment       12
  devtools         Developer Tools             8
  ...
```

**COUNT** 列是该分类中程序的数量(示例值)。
