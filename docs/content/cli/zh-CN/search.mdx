---
title: ossperks search
description: 通过关键词在名称、描述或提供方中搜索 OSS 福利项目。
---

## 用法

```
ossperks search <query> [options]
```

## 参数说明

| 参数  | 描述                                                                       |
| --------- | --------------------------------------------------------------------------------- |
| `<query>` | 搜索词（匹配名称、ID、描述、提供商、类别、标签或福利） |

## 选项

| 标志                    | 说明                 |
| ----------------------- | -------------------- |
| `-c, --category <slug>` | 限定在特定分类内搜索 |
| `--json`                | 输出原始 JSON        |
| `-h, --help`            | 显示帮助信息         |

## 示例

```bash

# 搜索“hosting”

ossperks search hosting

# 在 devtools 分类下搜索“credits”

ossperks search credits --category devtools

# 输出 JSON 格式

ossperks search vercel --json
```
