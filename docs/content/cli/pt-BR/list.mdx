---
title: ossperks list
description: Listar todos os programas de benefícios OSS disponíveis,
  opcionalmente filtrados por categoria.
---

## Uso

```
ossperks list [options]
```

**Alias:** `ossperks ls`

## Opções

| Flag                    | Descrição                                     |
| ----------------------- | --------------------------------------------- |
| `-c, --category <slug>` | Filtrar por slug de categoria (ex: `hosting`) |
| `--json`                | Exibir JSON bruto                             |
| `-h, --help`            | Exibir ajuda                                  |

## Exemplos

```bash

# Listar todos os programas

ossperks list

# Filtrar apenas programas de hospedagem

ossperks list --category hosting

# Saída JSON legível por máquina

ossperks list --json
```

## Saída

```
OSS Perks — 48 programas
────────────────────────────────

  1password              1Password for Open Source Projects
  algolia                Algolia for Open Source
  anthropic-claude       Anthropic Claude for Open Source
  ...
```
