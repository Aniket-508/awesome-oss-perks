---
title: ossperks categories
description: Lista todas as categorias de programas disponíveis e seus ids.
---

## Uso

```
ossperks categories [options]
```

**Alias:** `ossperks cats`

## Opções

| Opção        | Descrição        |
| ------------ | ---------------- |
| `--json`     | Exibir JSON puro |
| `-h, --help` | Mostrar ajuda    |

## Exemplos

```bash

# Listar todas as categorias

ossperks categories

# Saída em JSON

ossperks categories --json
```

## Saída

```
13 categorias
─────────────

  ID               NOME                    QUANTIDADE
  hosting          Hospedagem e Deploy        12
  devtools         Ferramentas de Desenvolvimento             8
  ...
```

A coluna **QUANTIDADE** é o número de programas naquela categoria (valores de exemplo).
