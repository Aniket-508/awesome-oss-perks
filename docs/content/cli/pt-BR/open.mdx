---

## Uso

```
ossperks open
```

## Descrição

Abre [ossperks.com](https://www.ossperks.com) no seu navegador padrão. Utiliza o comando apropriado para a plataforma (`open` no macOS, `xdg-open` no Linux, `start` no Windows).

Se o navegador não puder ser iniciado, a URL é exibida no terminal.

## Exemplos

```bash

# Abrir o site

ossperks open
```
