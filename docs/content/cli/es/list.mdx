---
title: ossperks list
description: Lista todos los programas de beneficios OSS disponibles,
  opcionalmente filtrados por categoría.
---

## Uso

```
ossperks list [options]
```

**Alias:** `ossperks ls`

## Opciones

| Opción                  | Descripción                              |
| ----------------------- | ---------------------------------------- |
| `-c, --category <id>` | Filtrar por ID de categoría (ej. `hosting`) |
| `--json`                | Mostrar JSON sin procesar                |
| `-h, --help`            | Mostrar ayuda                            |

## Ejemplos

```bash

# Listar todos los programas

ossperks list

# Filtrar solo programas de alojamiento

ossperks list --category hosting

# Salida JSON legible por máquina

ossperks list --json
```

## Salida

```
48 programas
────────────

  ID                NOMBRE                                        URL
  1password         1Password for Open Source Projects            https://ossperks.com/programs/1password
  algolia           Algolia for Open Source                         https://ossperks.com/programs/algolia
  anthropic-claude  Anthropic Claude for Open Source                https://ossperks.com/programs/anthropic-claude
  ...
```
