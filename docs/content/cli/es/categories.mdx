---
title: ossperks categories
description: Lista todas las categorías de programas disponibles y sus ids.
---

## Uso

```
ossperks categories [options]
```

**Alias:** `ossperks cats`

## Opciones

| Opción       | Descripción             |
| ------------ | ----------------------- |
| `--json`     | Genera JSON sin formato |
| `-h, --help` | Muestra ayuda           |

## Ejemplos

```bash

# Listar todas las categorías

ossperks categories

# Salida en JSON

ossperks categories --json
```

## Salida

```
13 categorías
─────────────

  ID               NOMBRE                  CANTIDAD
  hosting          Alojamiento e Implementación  12
  devtools         Herramientas de Desarrollo     8
  ...
```

La columna **CANTIDAD** es el número de programas en esa categoría (valores de ejemplo).
