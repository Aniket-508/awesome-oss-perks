title: ossperks open

---

## Uso

```
ossperks open
```

## Descripción

Abre [ossperks.com](https://www.ossperks.com) en tu navegador predeterminado. Utiliza el comando apropiado para la plataforma (`open` en macOS, `xdg-open` en Linux, `start` en Windows).

Si el navegador no se puede iniciar, la URL se imprime en la terminal en su lugar.

## Ejemplos

```bash

# Abrir el sitio web

ossperks open
```
