---

## 사용법

```
ossperks search <query> [options]
```

**별칭:** `ossperks find`, `ossperks s`

## 인수

| 인수  | 설명                                                                       |
| --------- | --------------------------------------------------------------------------------- |
| `<query>` | 검색어 (이름, ID, 설명, 제공자, 카테고리, 태그 또는 혜택과 일치) |

## 옵션

| 플래그       | 설명           |
| ------------ | -------------- |
| `--json`     | 원시 JSON 출력 |
| `-h, --help` | 도움말 표시    |

## 예제

```bash

# "hosting" 검색

ossperks search hosting

# 짧은 별칭 사용하기

ossperks s credits

# JSON 출력

ossperks search vercel --json
```

## 출력

```
"hosting" 검색 결과 — 4개 프로그램
────────────────────────────────────────────

  ID            NAME                            URL
  vercel        Vercel for Open Source          https://ossperks.com/programs/vercel
  netlify       Netlify Open Source Plan        https://ossperks.com/programs/netlify
  digitalocean  DigitalOcean Open Source        https://ossperks.com/programs/digitalocean
  cloudflare    Cloudflare for Open Source      https://ossperks.com/programs/cloudflare
```
