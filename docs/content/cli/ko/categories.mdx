## 사용법

```
ossperks categories [options]
```

**별칭:** `ossperks cats`

## 옵션

| 플래그       | 설명           |
| ------------ | -------------- |
| `--json`     | 원시 JSON 출력 |
| `-h, --help` | 도움말 표시    |

## 예제

```bash

# 모든 카테고리 나열

ossperks categories

# JSON 출력

ossperks categories --json
```

## 출력

```
13 categories
─────────────

  ID               NAME                    COUNT
  hosting          Hosting & Deployment       12
  devtools         Developer Tools             8
  ...
```

**COUNT** 열은 해당 카테고리에 속한 프로그램의 수입니다 (예시 값).
