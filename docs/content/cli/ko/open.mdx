---

## 사용법

```
ossperks open
```

## 설명

기본 브라우저에서 [ossperks.com](https://www.ossperks.com)을 엽니다. 플랫폼에 적합한 명령어를 사용합니다(macOS에서는 `open`, Linux에서는 `xdg-open`, Windows에서는 `start`).

브라우저를 실행할 수 없는 경우, URL이 터미널에 출력됩니다.

## 예제

```bash

# 웹사이트 열기

ossperks open
```
