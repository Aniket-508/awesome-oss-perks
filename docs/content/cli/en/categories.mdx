---
title: ossperks categories
description: List all available program categories and their ids.
---

## Usage

```
ossperks categories [options]
```

**Alias:** `ossperks cats`

## Options

| Flag         | Description     |
| ------------ | --------------- |
| `--json`     | Output raw JSON |
| `-h, --help` | Show help       |

## Examples

```bash
# List all categories
ossperks categories

# JSON output
ossperks categories --json
```

## Output

```
13 categories
─────────────

  ID               NAME                    COUNT
  hosting          Hosting & Deployment       12
  devtools         Developer Tools             8
  ...
```

The **COUNT** column is the number of programs in that category (example values).
