---
title: ossperks list
description: List all available OSS perk programs, optionally filtered by category.
---

## Usage

```
ossperks list [options]
```

**Alias:** `ossperks ls`

## Options

| Flag                    | Description                              |
| ----------------------- | ---------------------------------------- |
| `-c, --category <id>` | Filter by category id (e.g. `hosting`) |
| `--json`                | Output raw JSON                          |
| `-h, --help`            | Show help                                |

## Examples

```bash
# List every program
ossperks list

# Filter to hosting programs only
ossperks list --category hosting

# Machine-readable JSON output
ossperks list --json
```
