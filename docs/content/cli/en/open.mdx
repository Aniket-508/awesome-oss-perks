## Usage

```
ossperks open
```

## Description

Opens [ossperks.com](https://www.ossperks.com) in your default browser. Uses the platform-appropriate command (`open` on macOS, `xdg-open` on Linux, `start` on Windows).

If the browser cannot be launched, the URL is printed to the terminal instead.

## Examples

```bash
# Open the website
ossperks open
```
