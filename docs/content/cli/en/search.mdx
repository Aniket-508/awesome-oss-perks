---
title: ossperks search
description: Search OSS perk programs by keyword across names, descriptions, providers, tags, and perks.
---

## Usage

```
ossperks search <query> [options]
```

**Aliases:** `ossperks find`, `ossperks s`

## Arguments

| Argument  | Description                                                                       |
| --------- | --------------------------------------------------------------------------------- |
| `<query>` | Search term (matches name, slug, description, provider, category, tags, or perks) |

## Options

| Flag         | Description     |
| ------------ | --------------- |
| `--json`     | Output raw JSON |
| `-h, --help` | Show help       |

## Examples

```bash
# Search for "hosting"
ossperks search hosting

# Using the short alias
ossperks s credits

# JSON output
ossperks search vercel --json
```

## Output

```
Search results for "hosting" — 4 programs
────────────────────────────────────────────

  vercel                 Vercel for Open Source
  netlify                Netlify OSS Plan
  digitalocean           DigitalOcean Open Source
  cloudflare             Cloudflare for Open Source
```
