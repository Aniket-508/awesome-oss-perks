- **Fournisseur** : DeepSource
- **Catégorie** : sécurité
- **Site web** : [https://deepsource.com/pricing](https://deepsource.com/pricing)

## Avantages

- **Dépôts publics illimités** : Analysez un nombre illimité de dépôts publics gratuitement avec l'analyse statique, SAST et l'analyse d'infrastructure-as-code.
- **Revues de code automatisées** : Jusqu'à 1 000 revues de pull requests et 1 000 exécutions de formatage automatique de code par mois.
- **Couverture de code et détection de secrets** : Suivi intégré de la couverture de code et détection des secrets dans l'ensemble de votre base de code.

## Éligibilité

- Doit être un projet open source avec un dépôt public sur GitHub, GitLab ou Bitbucket.

## Comment postuler

1. Inscrivez-vous sur deepsource.com avec votre compte fournisseur Git.
2. Les dépôts publics sont automatiquement éligibles au plan Open Source gratuit.
