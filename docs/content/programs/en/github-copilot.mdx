## Perks

- **Free GitHub Copilot Pro**: Full access to GitHub Copilot Pro for AI-powered code completion, chat, and suggestions.

## Eligibility

- Must be a maintainer of a popular open-source repository on GitHub.
- Repository must meet GitHub's popularity/activity thresholds.

## How to apply

1. Eligible maintainers are automatically identified by GitHub.
2. Activate through GitHub Copilot settings.
