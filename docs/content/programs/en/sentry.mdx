- **Provider**: Sentry
- **Category**: monitoring
- **Duration**: No term limit
- **Website**: [https://sentry.io/for/open-source/](https://sentry.io/for/open-source/)

## Perks

- **5 Million Errors**: Track up to 5 million errors for free.
- **100 Million Transactions**: Performance monitoring with up to 100 million transactions.
- **10 GB Attachments**: Store up to 10 GB of error attachments.
- **100,000 Session Replays**: Replay user sessions to debug issues.

## Eligibility

- Must be an active open-source project.
- Project should use a recognized open-source license.
