## Perks

- **GitLab Ultimate Features**: Access to all GitLab Ultimate features including advanced CI/CD, security scanning, and compliance tools.
- **50,000 Compute Minutes**: 50,000 compute minutes per month for CI/CD pipelines.

## Eligibility

- Must be an open-source project.
- Project must be publicly available.
- Must welcome open collaboration and reuse.

## How to apply

1. Join the GitLab for Open Source program through the application page.
2. Ask questions in the GitLab Forum's open-source category.
