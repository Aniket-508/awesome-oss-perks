## Perks

- **6 Months of ChatGPT Pro**: Free ChatGPT Pro subscription with Codex access for coding, triage, review, and maintainer workflows.
- **API Credits**: Credits through the Codex Open Source Fund for PR review, automation, release workflows, or core open-source work.
- **Codex Security**: Conditional access to Codex Security for deeper code security scans and audits on qualified repositories.

## Eligibility

- No hard requirements like GitHub stars or monthly download thresholds.
- Must be a primary or core maintainer of an open-source project.
- Must demonstrate repository usage, ecosystem importance, and active maintenance.
- Must have a valid ChatGPT account.

## How to apply

1. Submit through OpenAI's official Codex for OSS form.
2. Applications are reviewed on a rolling basis.
3. Selected applicants are notified via email.
