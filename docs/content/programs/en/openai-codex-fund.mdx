- **Provider**: OpenAI
- **Category**: ai
- **Duration**: One-time grant (API credits)
- **Website**: [https://openai.com/form/codex-open-source-fund/](https://openai.com/form/codex-open-source-fund/)

## Perks

- **API Credits Grant**: Receive up to $25,000 in OpenAI API credits for your open source project.
- **Codex CLI Access**: Support for using Codex CLI and OpenAI models in your project workflows.

## Eligibility

- Must represent an open source project.
- Applications are reviewed on an ongoing basis.
- Funding decisions are made at OpenAI's sole discretion.

## How to apply

1. Fill out the application form on the OpenAI website.
2. Provide project details including GitHub repository and description.
3. Explain how you would use API credits for your project.
4. Applications are reviewed on an ongoing basis.
