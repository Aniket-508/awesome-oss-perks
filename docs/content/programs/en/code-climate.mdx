## Perks

- **Free Quality Platform**: Unlimited private and public contributors with full access to code coverage, maintainability analysis, and linting.
- **Auto-Formatting & Linting**: Automated code formatting and linting across your codebase with configurable rules.
- **Maintainability Analysis**: Track technical debt, code complexity, and maintainability metrics over time.

## Eligibility

- Must be an open source project with a public GitHub repository.

## How to apply

1. Sign up at codeclimate.com and add your public GitHub repository.
2. Open source repositories are automatically free forever.
