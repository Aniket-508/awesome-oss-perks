## Perks

- **Free Code Coverage**: Unlimited code coverage tracking for all public repositories with no time limits.
- **Multi-Language Support**: Supports C/C++, Go, Java, JavaScript/Node, Python, Ruby/Rails, PHP, .NET, and many more.
- **Unlimited Users**: All plans include unlimited users with instant sign-up through GitHub, Bitbucket, and GitLab.

## Eligibility

- Must be an open source project with a public repository on GitHub, GitLab, or Bitbucket.

## How to apply

1. Sign up at coveralls.io with your GitHub, Bitbucket, or GitLab account.
2. Public repositories are automatically free forever — no credit card required.
