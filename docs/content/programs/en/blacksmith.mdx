## Perks

- **Free CI Minutes**: Free access to Blacksmith's high-performance GitHub Actions runners for qualifying open source projects.

- **4x Faster Cache Downloads**: Significantly faster cache downloads for quicker builds and shorter feedback loops.

## Eligibility

- Actively maintained public repository.
- Permissive open source license.
- Clear community usage.

## How to apply

1. Apply through the OSS Program section on the Blacksmith pricing page.
