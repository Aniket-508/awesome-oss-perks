## Vorteile

- **Kostenlose CI-Minuten**: Kostenloser Zugang zu Blacksmiths leistungsstarken GitHub Actions Runnern für qualifizierte Open-Source-Projekte.
- **2x schnellere Hardware**: Runner mit 2x schnellerer Hardware im Vergleich zu den Standard-Runnern von GitHub, wodurch sich die CI-Laufzeit halbiert.
- **4x schnellere Cache-Downloads**: Deutlich schnellere Cache-Downloads für kürzere Build-Zeiten und schnellere Feedback-Schleifen.

## Berechtigung

- Aktiv gepflegtes öffentliches Repository.
- Permissive Open-Source-Lizenz.
- Erkennbare Community-Nutzung.

## So bewerben Sie sich

1. Bewerben Sie sich über den Bereich OSS Program auf der Preisseite von Blacksmith.
2. Bewerbungen werden vom Blacksmith-Team geprüft.
