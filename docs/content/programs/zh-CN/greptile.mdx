- **提供商**：Greptile
- **类别**：人工智能
- **网站**：[https://www.greptile.com/open-source](https://www.greptile.com/open-source)

## 权益

- **免费 AI 代码审查**：符合条件的开源仓库可 100% 免费使用 Greptile。提供基于完整代码库理解的自动化 GitHub PR 审查。
- **代码上下文与自定义规则**：AI 理解您的代码库上下文,支持自定义规则并从您的仓库中学习。

## 资格要求

- 采用 MIT、Apache 或 GPL 许可证的开源项目。
- 非商业项目。
- 仓库所有者或管理员能够为仓库安装应用。

## 申请方式

1. 通过 Greptile 开源项目计划页面提交申请,需提供电子邮件、姓名和仓库链接。
2. 确认您是仓库的所有者或管理员,并能够安装应用。
