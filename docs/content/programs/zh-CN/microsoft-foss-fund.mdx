---

- **提供方**：Microsoft
- **类别**：资金支持
- **周期**：季度资助轮次
- **网站**：[https://github.com/microsoft/foss-fund](https://github.com/microsoft/foss-fund)

## 福利

- **季度赞助**：每季度向一个或多个开源项目分发最高 12,500 美元。
- **员工驱动选择**：项目由积极为开源社区做出贡献的 Microsoft 员工选择。
- **资金使用灵活**：受助者可以根据项目需求灵活使用资金。

## 资格要求

- 该开源项目必须被 Microsoft 使用。
- 项目必须拥有 OSI 批准的开源许可证。
- 必须具备符合采购和法律要求的资金接收方式（首选 GitHub Sponsors）。
- 项目不能由 Microsoft 员工拥有。
- 鼓励优先考虑多样性、公平性和包容性的项目。

## 如何申请

1. Microsoft 员工或实习生可通过提名表格提名项目。
2. 项目由"开源先锋"策展小组审核。
3. 为开源项目做出贡献的 Microsoft 员工投票选择资助项目。
4. 入选项目直接获得赞助资金。
